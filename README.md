## 🎯 プロジェクト概要

TIL リポジトリは、日々学んだこと・気づきをまとめて記録するための場所です。  
小さな知見であっても「学び」を蓄積し続けることで、自身の成長を可視化できます。

---

## 📂 ディレクトリ構成

```text
TIL/
├── YYYY-MM-DD/           # 日付ごとのフォルダ (例: 2025-07-19)
│   ├── topic1.md         # 学んだトピック1
│   └── topic2.md         # 学んだトピック2
├── assets/               # 画像や図などのリソース
├── template.md           # 新しいエントリ作成用のテンプレート
└── README.md             # 本ファイル
